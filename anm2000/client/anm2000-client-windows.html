Instalação do Cliente de Ativação da OLT Fiberhome

O objetivo deste tutorial é instalar o software ANM-2000 no Windows.

Índice

  1. Baixar o instalador
  2. Instalar o programa
  3. Acessar o programa

Passo 1:

Baixar o instalador

Abra o link do Google Drive https://drive.google.com/drive/folders/1T2exs2cIjmW61po6-rs9fd4hJ_37fJpo e baixe o arquivo Anm2000_Client_2014_11_24_V6.exe .

Passo 2:

Instalar o programa

Após fazer o download, execute o arquivo de instalação Anm2000_Client_2014_11_24_V6.exe . Dê permissão se for pedido e agurde ele carregar.

Após carregado o instalador, clique em Next.

Em seguida, Yes.

Preencha o User Name e a Company Name como Minasnet.

Selecione o idioma English.

ATENÇÃO! Pode ocorrer um bug durante a instalação neste momento se não for feito o procedimento corretamente.

Verifique se a pasta de destino da instalação está em C:\AEMS_Client. Se não estiver, clique em Browse.

Após clicar em Browse, pode aparecer uma mensagem de CD. Ignore ela clicando em Cancelar.

Após ignorar a mensagem de CD, mude o campo Path digitando C:\AEMS_Client (normalmente é só trocar a letra errada por C). Feita a modificação, clique em OK

Certifique-se de que as configurações exibidas após o procedimento anterior estão conforme a tela a seguir. Clique em Next para instalar e agurde.

A instalação pode demorar um pouco.

Após a instalação, aparecerá a mensagem pedindo para reiniciar o computador. Não é necessário, somente ignore clicando em OK.

Em seguida, selecione No para não reiniciar o computador e finalize a instalação em Finish.

Passo 3:

Acessar o programa

Após a instalação, abra o Anm2000.

Na tela de login, clique nos [...] para salvar o IP do servidor.

Clique em Add.

Na linha adicionada, coloque o IP 10.6.6.73, conforme a tela a seguir. Clique em OK.

Agora, basta fazer login no Anm2000 para acessar o programa. O usuário e a senha são 1.

PRONTO!
