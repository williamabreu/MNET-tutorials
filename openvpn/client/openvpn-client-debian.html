Configuração de cliente OpenVPN em Debian

O objetivo deste tutorial é conectar um cliente OpenVPN em Debian/Linux. Todo o procedimento é feito usando terminal.

Índice

  1. Instalar OpenVPN
  2. Obter arquivo de configuração do cliente
  3. Conectar-se à VPN
  4. Desconectando-se da VPN

<username> é expressão regular para ser substituída pelo seu nome de usuário atribuído pelo sysadmin. Por exemplo, seja o usuário william, onde ocorrer a expressão <username>.ovpn deve ser feita a troca por william.ovpn.

Passo 1:

Instalar OpenVPN

$ sudo apt update
$ sudo apt install openvpn

Passo 2:

Obter arquivo de configuração do cliente

Obtenha o arquivo de acesso com o administrador da VPN. O arquivo tem o formato <username>.ovpn.

Com posse do arquivo, navegue até o diretório onde se encontra e proceda com o seguinte:

$ sudo chmod 700 <username>.ovpn
$ sudo mv <username>.ovpn /etc/openvpn/client/

Passo 3:

Conectar-se à VPN

Execute o seguinte comando para conectar-se à VPN:

$ sudo openvpn --config /etc/openvpn/client/<username>.ovpn

Será pedido login e senha no prompt.

Passo 4:

Desconectando-se da VPN

Para desconectar-se, basta dar ^C. Cuidado, pois se fechar o terminal a tarefa é encerrada.
